Hur man kan använda Github som system i Skolan

Github är ett så kallat "Socialt nätverk för utvecklare", där folk kan hjälpa till att jobba på olika Open Source projekt.

Här står det hur man kan använda det för uppgifter i skolan, ungefär som Edwise.

Hur man använder det som Elev

Steg 1 - Skaffa ett Github Konto

Gå in på https://github.com, och använd formuläret på framsidan.

OBS! Se till att använda en fungerande email-adress.

Steg 2 - Kopiera grundarkivet1 från Gruppen PRRPRR01TE2 till din egen användare

Gå in på https://github.com/PRRPRR01TE2/uppgifter.
Tryck på "Fork" 2 uppe i det högra hörnet.

Ett eget arkiv, med samma namn, kommer nu att skapas på din användare.

Steg 3 - Ladda ner & installera Github-klienten

Gå in på http://windows.github.com och ladda ner applikationen.

Följ instruktionerna för att installera.
